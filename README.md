# CarEcommerce

## OverView
車の情報を見ることができる簡易的なECサイトです。メーカー・価格・新着順に並び替えることができます。コンピュータサイエンス学習サービス[Recursion](https://recursionist.io/)のVue.jsのアウトプットプロジェクト（Car Ecommerce）として取り組みました。
